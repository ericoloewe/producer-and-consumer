# Producer and consumer made using Erlang

An escript

## Tecnologies
- [erlang](https://www.erlang.org/)
- [rebar3](https://www.rebar3.org/)

## Developing

Build
-----

    $ rebar3 escriptize

Run
---

    $ _build/default/bin/producer_and_consumer
